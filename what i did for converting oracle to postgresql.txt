Key Changes Made:
1. Data Types:
	NUMBER(19) → BIGSERIAL (auto-increment) or BIGINT (foreign keys)
	NUMBER(18,2) → NUMERIC(18,2)
	NUMBER(6,5) → NUMERIC(6,5)
	NUMBER(10) → INTEGER
	NUMBER(1) → BOOLEAN (for auto_renew)
	VARCHAR2 → VARCHAR or TEXT (for details fields)
	TIMESTAMP → TIMESTAMPTZ (timestamp with timezone)
2. Constraints:
	Moved PRIMARY KEY inline with column definition
	REFERENCES instead of separate FOREIGN KEY clauses (cleaner syntax)
	UNIQUE moved inline where applicable
	All CHECK constraints preserved
3. Defaults:
	CURRENT_TIMESTAMP → NOW() (both work, but NOW() is more PostgreSQL-idiomatic)
	DEFAULT values preserved
